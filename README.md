🧬 Digital Twin Simulation of Antiviral Clinical Trials

This repository contains code for a digital twin framework that simulates antiviral clinical trials using a combination of:

- Within-host viral dynamics models
- PK/PD (pharmacokinetic/pharmacodynamic) modeling
- Virtual patient generation based on population parameters
- Trial-specific endpoint evaluation and statistical analysis


🎯 Project Objective

The goal of this project is to:

Reproduce and evaluate clinical trial outcomes by simulating virtual patient populations under real-world trial designs.

Specifically, this framework:

- Simulates viral load trajectories for virtual patients
- Recreates trial designs (sample size, patient characteristics, timing)
- Applies endpoint definitions aligned with original studies
- Performs statistical analyses to determine trial outcomes (e.g., significance)
